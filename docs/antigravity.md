# <img src="https://antigravity.google/assets/image/brand/antigravity-icon__full-color.png" alt="Antigravity logo" align="right" width="96" style="margin-top: 1ex; border-left: 0.1em solid transparent;"> Antigravity

Google [Antigravity](https://antigravity.google/docs/home) is an Integrated Development Environment designed for architecting and managing autonomous AI agentic systems. It provides a high-level visual interface for designing, orchestrating, and monitoring multiple agents as they collaborate on complex tasks. Think of it as a control tower for your AI workforce, allowing you to build and debug agent interactions rather than individual lines of code.

- ↪️ On Debian, run recipe: `just install-apt-packages`
