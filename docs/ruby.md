# Ruby

[Ruby](https://ruby-lang.org/) is an open-source, fully object-oriented, and interpreted programming language that prioritizes simplicity and developer productivity. Developed in the mid-1990s by Yukihiro "Matz" Matsumoto, its syntax is designed to be natural and easy to read, often resembling plain English.

## Installation

- [Ruby is installed via mise](https://mise.jdx.dev/lang/ruby.html)
- [Rails](https://guides.rubyonrails.org/getting_started.html) framework: there is a task to `install_rails`
- [Jekyll](https://jekyllrb.com/) was the first [static site generator](https://jamstack.org/generators/) used by GitHub Pages; there is a task to `install_jekyll`
- [RubyGems](https://guides.rubygems.org/command-reference/) packages
