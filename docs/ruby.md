# Ruby

[Ruby](https://ruby-lang.org/) is an open-source, fully object-oriented, and interpreted programming language that prioritizes simplicity and developer productivity. Developed in the mid-1990s by Yukihiro "Matz" Matsumoto, its syntax is designed to be natural and easy to read, often resembling plain English.

## Installation

- ↪️ Recipe to install Ruby [with Mise](https://mise.jdx.dev/lang/ruby.html): `just install-ruby`
- Ruby package manager: [RubyGems](https://guides.rubygems.org/command-reference/) packages
- ↪️ Recipe to install its [Rails](https://guides.rubyonrails.org/getting_started.html) framework: `install-rails`
- ↪️ Recipe to install [Jekyll](https://jekyllrb.com/) ([static site generator](https://jamstack.org/generators/)): `install-jekyll`
