# Haskell

> [Haskell](https://www.haskell.org/) is a purely functional, statically typed programming language known for its strong emphasis on code safety and correctness. Its core features include **type inference**, which catches errors at compile time, and **lazy evaluation**, allowing for the creation of efficient, modular code and infinite data structures. Haskell's design encourages developers to write highly declarative and maintainable programs by avoiding side effects. It is widely used in academia and industry for tasks requiring high reliability, such as in finance, compiler design, and data analysis.
