# Goose

[Goose](https://block.github.io/goose/docs/) is an agentic AI coding assistant from Block designed to automate complex development tasks from start to finish. It can understand high-level instructions, create a plan, and execute changes across your codebase, acting as an autonomous developer to help you build and refactor software.

- ↪️ Run recipe to install: `just install-goose`
