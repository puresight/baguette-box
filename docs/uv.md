# UV

[UV](https://docs.astral.sh/uv/getting-started/features/), the unified tool that replaces utilities in the Python ecosystem (pip, pip-tools, pyenv, virtualenv, pipx, poetry, twine), installs [Python](https://www.python.org/).

## Applications

Here are some Python apps you can check out:

- [Pgcli](https://pypi.org/project/pgcli/) an enhanced CLI for PostgreSQL with auto-completion and syntax highlighting `uv tool install pgcli`
- [Mycli](https://pypi.org/project/mycli/) an enhanced CLI for MySQL with auto-completion and syntax highlighting `uv tool install mycli`
- [JupyterLab](https://jupyter.org/) (uv): A web-based interactive development environment for Jupyter Notebooks `uv tool install jupyterlab`
- [LocalStack](https://pypi.org/project/localstack/) - popular dev tool for emulating AWS cloud services (including databases like DynamoDB) locally `uv tool install localstack`

## Libraries

Here are some Python libraries:

- [Streamlit](https://streamlit.io/) - a system/library used to turn data scripts into shareable web apps in minutes, good for database dashboards `uv add pandas streamlit`
