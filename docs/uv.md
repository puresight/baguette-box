# UV

[UV](https://docs.astral.sh/uv/getting-started/features/), the unified tool that replaces utilities in the Python ecosystem (pip, pip-tools, pyenv, virtualenv, pipx, poetry, twine), installs [Python](https://www.python.org/).

- ↪️ Run recipe to install: `just install-uv`

## Applications

Find apps in the [PyPI](https://pypi.org/) Python Package Index repository of software for the Python programming language.

Here are some you might like:

- [Pgcli](https://pypi.org/project/pgcli/) an enhanced CLI for PostgreSQL with auto-completion and syntax highlighting `uvx pgcli`
- [Mycli](https://pypi.org/project/mycli/) an enhanced CLI for MySQL with auto-completion and syntax highlighting `uvx mycli`
- [JupyterLab](https://jupyter.org/) (uv): A web-based interactive development environment for Jupyter Notebooks `uvx jupyterlab`
- [LocalStack](https://pypi.org/project/localstack/) - popular dev tool for emulating AWS [cloud](./dev-server.md) services (including databases like DynamoDB) locally `uvx localstack`
- [LiteCLI](https://litecli.com/) is a command line interface for SQLite with auto-completion and syntax highlighting `uvx litecli`
- [IRedis](https://iredis.io/) is a terminal client for redis with auto-completion and syntax highlighting. `uvx iredis`

After the trial, if you want to keep it, swap `uv tool install` in place of `uvx` above.
