# dotnet

Microsoft [.NET](https://dotnet.microsoft.com/) is a free, open-source developer platform created by Microsoft for building many different types of applications. It provides a collection of tools, programming languages, and libraries that allow devs to create apps for Windows desktop or console, ASP.NET web, etc.
