# <figure style="float: right; margin: 0.4ex 0.5em; line-height: 1; text-align: center; font-size: 80px">📥</figure> AppImage

[AppImage](https://appimage.org/) is a portable software format for Linux that bundles an application and its dependencies into a single file, allowing it to run on almost any distribution without installation or root permissions. Apps require only that the file be made executable, for example: `chmod +x AFFiNE-x.x.x-linux.AppImage`
