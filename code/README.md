# Code

This folder holds all information for installing and configuring
Microsoft Visual Studio [Code](https://code.visualstudio.com/) IDE,
including the [`code.dep`](./code.dep) list of
[extension](https://marketplace.visualstudio.com/vscode)
dependencies to install such as

- [Gemini Code Assist](https://marketplace.visualstudio.com/items?itemName=Google.geminicodeassist)
- [Roo Code](https://roocode.com/)

Here you will find JSON files with configuration updates
that shall be applied (in a shallow merge)
when the Code IDE is installed or configured, including

- `argv.json`
- `user-settings.json`
